# Plataformas de WhatsApp Business

## Comparativo

| Critério | Evolution API | OpenClaw | Tyxter Studio |
|----------|--------------|----------|---------------|
| Custo | Gratuito (self-hosted) | Por mensagem | Mensal fixo |
| Setup | Médio | Simples | Simples |
| Escala | Limitada | Alta | Muito Alta |
| SLA | Sem garantia | 99.5% | 99.9% |
| Suporte | Comunidade | Email | Dedicado |

## Quando usar cada um

### Evolution API
- Desenvolvimento local
- Testes de integração
- MVP com baixo volume

### OpenClaw
- Produção com volume médio
- Sem equipe de infra
- Marketplace de bots

### Tyxter Studio
- Produção enterprise
- Alto volume de mensagens
- SLA crítico
