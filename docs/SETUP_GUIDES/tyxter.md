# Setup Tyxter Studio

## Pré-requisitos
- Conta Tyxter Studio (plano Business ou superior)
- Número WhatsApp Business verificado

## Configuração

1. Acesse https://studio.tyxter.com
2. Crie um novo projeto
3. Em "Integrações", configure o webhook: `https://seu-dominio.com/webhook/tyxter`
4. Copie a API Key e o Webhook Secret

## Variáveis de Ambiente

```
TYXTER_API_KEY=sua-chave-aqui
TYXTER_WEBHOOK_SECRET=seu-secret-aqui
```

## Recursos Exclusivos Tyxter

- **Botões interativos**: Menus de seleção nativos do WhatsApp
- **Listas**: Exibição de opções em formato lista
- **Templates**: Mensagens aprovadas pela Meta para notificações
