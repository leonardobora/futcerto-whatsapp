# Setup OpenClaw

## Pré-requisitos
- Conta no OpenClaw
- Número WhatsApp Business aprovado

## Configuração

1. Acesse o dashboard OpenClaw
2. Crie um novo bot
3. Configure o webhook: `https://seu-dominio.com/webhook/openclaw`
4. Copie a API Key para `OPENCLAW_API_KEY`

## Variáveis de Ambiente

```
OPENCLAW_API_KEY=sua-chave-aqui
OPENCLAW_WEBHOOK_SECRET=seu-secret-aqui
```
